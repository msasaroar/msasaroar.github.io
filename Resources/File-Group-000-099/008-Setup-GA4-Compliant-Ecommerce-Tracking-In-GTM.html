Setup Google Analytics 4(As Per GA4 Schema) Ecommerce Tracking with Google Tag Manager

Step-by-Step Guide

  1. Configure GA4 Property

    • Log in to your Google Analytics account and create a GA4 property.
    • Copy the Measurement ID (e.g., G-XXXXXXXXXX).

  2. Set Up Google Tag Manager (GTM)

    • Log in to Google Tag Manager and create a container for your website.
    • Add the GTM container code to your website's <head> and <body> sections.

  3. Create a GA4 Configuration Tag

    • In GTM, click Tags > New.
    • Choose GA4 Configuration as the tag type.
    • Enter your GA4 Measurement ID.
    • Set the trigger to All Pages.
    • Save and publish the tag.

  4. Enable Enhanced eCommerce Features

    • Go to your GA4 property settings.
    • Navigate to Enhanced Measurement and enable eCommerce tracking.

  5. Implement Data Layer for eCommerce

    Add a structured data layer on your website to capture eCommerce events:

    {
"event": "purchase",
"ecommerce": {
  "transaction_id": "T12345",
  "value": 150.00,
  "currency": "USD",
  "items": [
    {
      "item_name": "Product A",
      "item_id": "SKU12345",
      "price": 50.00,
      "quantity": 2
    },
    {
      "item_name": "Product B",
      "item_id": "SKU67890",
      "price": 50.00,
      "quantity": 1
    }
  ]
}
}

  6. Create GA4 Event Tags in GTM

    • In GTM, click Tags > New.
    • Select GA4 Event as the tag type.
    • Configure the tag with the following settings:
      • Event Name: Match the event in your data layer (e.g., purchase).
      • Event Parameters: Map data layer variables to GA4 parameters (e.g., value, currency).
    • Set the trigger to fire on specific events (e.g., purchase).
    • Save and publish the tag.

  7. Test and Debug

    • Use GTM's Preview Mode to test your setup.
    • Verify events in GA4's DebugView.

  8. Publish Your GTM Changes

    • Once tested, publish the container in GTM.
