Detailed Activities For Setting Up Conversion Tracking In Pinterest Pixel With GTM


1. Obtain the Pinterest Pixel ID
- Log in to your Pinterest Ads Manager.
- Navigate to "Ads" > "Conversions" and select "Pinterest Tag".
- Copy the unique Pinterest Pixel ID.

2. Set Up the Pinterest Base Code in GTM
- Open Google Tag Manager and create a new tag.
- Choose "Custom HTML" as the tag type.
- Paste the Pinterest base code inside the tag. The base code format is:
    ```html
    <script>
    !function(e){if(!window.pintrk){window.pintrk=function(){window.pintrk.queue.push(arguments)};var n=window.pintrk;n.queue=[],n.version="3.0";var t=document.createElement("script");t.async=!0,t.src=e;var r=document.getElementsByTagName("script")[0];r.parentNode.insertBefore(t,r)}}("https://s.pinimg.com/ct/core.js");
    pintrk('load', 'YOUR_PIXEL_ID');
    pintrk('page');
    </script>
    ```
- Replace `'YOUR_PIXEL_ID'` with your actual Pinterest Pixel ID.
- Set the trigger to "All Pages" to fire the base code on every page.
- Save the tag.

3. Create a Conversion Event in GTM
- Create a new tag in GTM.
- Select "Custom HTML" as the tag type.
- Paste the Pinterest event tracking script inside the tag.
- Modify the script to match the conversion event you want to track.
- Example for a "Lead" event:
    ```html
    <script>
    pintrk('track', 'lead');
    </script>
    ```
- Save the tag.

4. Set Up Standard Events or Custom Events
- Click "Triggers" in GTM and create a new trigger.
- Choose an appropriate trigger type (e.g., Click, Form Submission, Page View).
- Define trigger conditions, such as:
    - "Page URL contains /thank-you" (for form submission tracking).
    - "Click ID equals 'purchase-button'" (for purchase button clicks).
- Assign the trigger to the Pinterest event tag.
- Save the trigger.

5. Test in GTM Preview Mode
- Click "Preview" in Google Tag Manager.
- Enter your website URL and perform the conversion action.
- Verify that the Pinterest event tag fires in the GTM Debug Console.

6. Verify in Pinterest Ads Manager
- Open Pinterest Ads Manager and go to "Conversions".
- Check if the event appears in the event history.
- Use the Pinterest Tag Helper Chrome Extension to confirm event tracking.

7. Publish GTM Container
- If everything works correctly, go back to Google Tag Manager.
- Click "Submit" and publish the updated GTM container.
- Document the changes for reference.

8. Monitor and Optimize
- Regularly check Pinterest Ads Manager for accurate conversion tracking.
- Use the Pinterest Tag Helper tool to debug any tracking issues.
- Adjust event parameters or triggers based on data insights.
- Compare Pinterest conversion data with other analytics tools (e.g., GA4) to ensure consistency.
