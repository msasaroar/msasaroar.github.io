GA4 Compliant Events

1. Automatically Collected Events

  • first_visit: Triggered the first time a user visits the site.
  • session_start: Logged at the start of a new session.
  • user_engagement: Tracked when the app goes to the background or after a session timeout.

2. Enhanced Measurement Events

  • page_view: Logged whenever a page is viewed.
  • scroll: Triggered when a user scrolls 90% of the page.
  • click: Tracked for outbound links by default.
  • file_download: Triggered when users download a file.
  • video_start, video_progress, video_complete: Tracked for embedded video engagement.
  • view_search_results: Logged when a user performs a search on the site.

3. Recommended Events

E-Commerce and Retail

  • add_to_cart: When an item is added to the shopping cart.
  • add_to_wishlist: When an item is added to a wishlist.
  • begin_checkout: When a user starts the checkout process.
  • purchase: For a completed purchase.
  • refund: When a transaction is refunded.
  • view_item: When a user views a specific product.
  • view_item_list: When a user views a list of items.

Travel and Hospitality

  • select_item: When a user selects a travel item like a flight or hotel.
  • add_payment_info: When a user adds payment details.

Media and Entertainment

  • sign_up: When a user signs up for an account.
  • login: When a user logs in.
  • subscribe: For paid subscriptions.
  • share: When users share content via social or other platforms.

4. Custom Events

When none of the predefined events match your specific tracking needs, you can define custom events. Examples include:

  • form_submission
  • video_mute
  • coupon_applied
