GA4 Subscribe Event Schema

Schema Details

Parameter Name Data Type Description
plan String The name of the subscription plan (e.g., "Basic", "Premium").
user_id String The unique identifier for the user.
success Boolean Indicates whether the subscription was successful.

Example JSON for the Subscribe Event

{
  "event_name": "subscribe",
  "parameters": {
    "plan": "Premium",
    "user_id": "user123",
    "success": true
  }
}
