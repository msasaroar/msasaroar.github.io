GA4 Product Interaction Events Schema

Schema Details for Product Interaction Events

Event Name Description
view_item Tracks when a user views a product or item.
select_item Tracks when a user selects a product from a list or category.
add_to_cart Tracks when a user adds a product to their shopping cart.
remove_from_cart Tracks when a user removes a product from their shopping cart.
begin_checkout Tracks when a user starts the checkout process.
purchase Tracks when a user completes a purchase.
add_to_wishlist Tracks when a user adds a product to their wishlist.

Common Parameters

Parameter Name Data Type Description
value Float The monetary value associated with the event.
currency String The currency used (e.g., "USD").
items Array An array of items associated with the event.
coupon String The coupon code applied (if any).
transaction_id String A unique identifier for the transaction (for purchase events).

Items Parameter Schema

Field Name Data Type Description
item_id String The unique identifier of the item.
item_name String The name of the item.
quantity Integer The number of units of the item.
price Float The price of a single unit of the item.
item_brand String The brand of the item.
item_category String The category of the item.
item_variant String The variant of the item (e.g., color, size).
discount Float The discount applied to the item, if any.
