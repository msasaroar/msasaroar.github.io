Demonstration : Refund Event with Data Layer

Refund Event is used to track refund transactions in Google Analytics 4.
By capturing this event, businesses can analyze refund trends, identify issues, and improve customer satisfaction.
It involves sending data such as transaction ID and refund value to GA4 for accurate reporting.
