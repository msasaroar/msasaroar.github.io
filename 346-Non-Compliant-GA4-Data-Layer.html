Non-Compliant GA4 Data Layer

This page defines a sample data layer that does not comply with the GA4 schema. The following issues exist:

  • transactionId should be transaction_id
  • totalValue should be value
  • currencyCode should be currency
  • name should be item_name
  • id should be item_id
  • unit_price should be price

Defined Data Layer

{
  "event": "purchase",
  "ecommerce": {
    "transactionId": "T12345",
    "totalValue": 150.00,
    "currencyCode": "USD",
    "items": [
      {
        "name": "Product A",
        "id": "SKU12345",
        "unit_price": 50.00,
        "quantity": 2
      },
      {
        "name": "Product B",
        "id": "SKU67890",
        "unit_price": 50.00,
        "quantity": 1
      }
    ]
  }
}
