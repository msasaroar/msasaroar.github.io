GA4 Refund Event Schema

Schema Details

Parameter Name Data Type Description
transaction_id String The unique identifier of the transaction being refunded.
value Float The total refund amount (may not equal the full transaction value).
currency String The currency used for the refund (e.g., "USD").
refund_reason String The reason for the refund (e.g., "Product Defect").
items Array An array of refunded items, where each item is an object containing details.

Items Parameter Details

Field Name Data Type Description
item_id String The unique identifier of the item being refunded.
item_name String The name of the item.
quantity Integer The number of units refunded for the item.
price Float The price of a single unit of the item.
item_brand String The brand associated with the item.
item_category String The category associated with the item.
item_variant String The variant of the item (e.g., color or size).

Example JSON for the Refund Event

{
  "event_name": "refund",
  "parameters": {
    "transaction_id": "T12345",
    "value": 45.00,
    "currency": "USD",
    "refund_reason": "Product Defect",
    "items": [
      {
        "item_id": "P001",
        "item_name": "Smartphone",
        "quantity": 1,
        "price": 45.00,
        "item_brand": "BrandX",
        "item_category": "Electronics",
        "item_variant": "Blue"
      }
    ]
  }
}
