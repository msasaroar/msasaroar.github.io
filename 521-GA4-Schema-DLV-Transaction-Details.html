Data Layer Variable: Transaction Details

Transaction Details Schema

Key Data Type Description
transaction_id String A unique identifier for the transaction.
affiliation String The store or affiliation associated with the transaction.
value Float The total value of the transaction (including taxes and shipping).
currency String The currency used for the transaction (e.g., "USD").
tax Float The total tax amount for the transaction.
shipping Float The shipping cost for the transaction.
coupon String The coupon code applied to the transaction, if any.
items Array An array of items included in the transaction, where each item is an object containing details.

Items Parameter Details

Key Data Type Description
item_id String The unique identifier of the item.
item_name String The name of the item.
quantity Integer The number of units purchased for the item.
price Float The price of a single unit of the item.
item_brand String The brand associated with the item.
item_category String The category associated with the item.
item_variant String The variant of the item (e.g., color or size).
discount Float The discount applied to the item, if any.
