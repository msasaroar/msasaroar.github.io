DOM Ready Trigger Demonstration

About the Page

This page demonstrates how to create a 'DOM Ready Trigger' using Google Tag Manager (GTM).

Steps to Create the DOM Ready Trigger and Tag:

  1. Log in to your Google Tag Manager account.
  2. Go to the workspace containing your container.
  3. Click on Triggers in the left navigation panel.
  4. Click on New to create a new trigger.
  5. Name your trigger: DOM Ready Trigger.
  6. Select Trigger Type as DOM Ready.
  7. Save the trigger.
  8. Next, create a tag that fires on this trigger:
    • Go to Tags in the left navigation panel.
    • Click New.
    • Name your tag: DOM Ready Tag.
    • Choose Tag Configuration and select a tag type (e.g., Custom HTML).
    • Add your desired script or content.
    • Under Triggering, choose the DOM Ready Trigger you created.
  9. Click Save and submit the container changes.

Now the tag will fire when the DOM is fully loaded!
