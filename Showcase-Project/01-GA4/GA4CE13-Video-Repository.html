Why Is Repository Video Engagement Tracking a Custom Event in GTM?

1. Repository videos (such as self-hosted files or those embedded from third-party platforms other than YouTube) are not automatically tracked by Google Tag Manager (GTM) or Google Analytics 4 (GA4).

2. GTM does not include built-in triggers for detecting video play, pause, or complete actions on custom or repository video players.

3. You need to use a Custom HTML Tag that runs JavaScript to detect these user actions on the video.

4. Then you must configure Custom Event Triggers in GTM that respond when these JavaScript events are pushed to the dataLayer.
