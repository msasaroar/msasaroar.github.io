Why Is Repository Video Engagement Tracking a Custom Event in GTM?

1. Repository videos (self-hosted or non-YouTube) are not tracked automatically by GTM/GA4.
2. GTM lacks built-in triggers for play/pause/complete on such videos.
3. You need a Custom HTML Tag with JS to detect these events.
4. Custom Event triggers in GTM listen for these JS events pushed to dataLayer.
