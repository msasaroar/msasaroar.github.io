GTM Basics: Tags, Triggers, Variables & Events

üìå What is Google Tag Manager (GTM)?

Google Tag Manager is a free tool from Google that allows marketers and developers to manage and deploy marketing tags (snippets of code or tracking pixels) on your website without modifying the code directly.

üè∑Ô∏è Tag

A Tag is a piece of code that sends data to platforms like Google Analytics, Facebook Pixel, or others.

  • Example: GA4 Configuration Tag, GA4 Event Tag
  • Fires when the Trigger condition is met
  • Created under the "Tags" section in GTM

üéØ Trigger

A Trigger tells GTM when or under what condition a tag should fire.

  • Types: Page View, Click, Form Submission, Timer, Custom Event, etc.
  • Example: Fire tag only on page URL contains thank-you
  • Can include filters/conditions

üîß Variable

Variables are used to capture dynamic values, like page URL, click text, or form field values.

  • Built-in Examples: Page URL, Page Path, Referrer
  • User-defined Examples: DOM Element, Data Layer Variable
  • Variables make your triggers smarter

‚ö° Event

Events are interactions users make on your site. GTM uses dataLayer.push to track events and trigger tags.

  • Example: dataLayer.push({event: 'form_submit'})
  • Can be Custom Events or automatic (like page views)
  • Important for GA4 event-based tracking

üìö Summary

  • Tag: Sends data (e.g., GA4, Ads)
  • Trigger: When to fire the tag
  • Variable: Dynamic values for triggers/tags
  • Event: User interaction or behavior sent to GTM
