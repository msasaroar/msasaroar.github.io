GA4 Event Name: disqualify_lead

Event Definition:

This event is used to track leads that were disqualified (e.g., unqualified for services/products).

GA4 Event Parameters

  • lead_id (string) - Unique identifier for the lead
  • disqualification_reason (string) - Why the lead was disqualified (e.g., "budget", "location")
  • lead_type (string) - Type of lead (e.g., "B2B", "B2C")
  • disqualified_by (string) - Who marked the lead as disqualified (e.g., agent name or system)
  • currency (string) – Currency code (e.g., "USD")
  • value (number) – Potential value of the disqualified lead
