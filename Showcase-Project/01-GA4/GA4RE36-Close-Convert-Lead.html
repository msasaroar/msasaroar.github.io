GA4 Event: close_convert_lead

This event marks a lead successfully converted into a customer.

Required/Optional Parameters:

  • lead_type (optional): e.g., "B2B"
  • value (optional): e.g., 1200
  • currency (optional): e.g., "USD"
  • transaction_id (optional): Unique ID

Data Pushed to Data Layer:

dataLayer.push({
  event: "close_convert_lead",
  lead_type: "B2B",
  value: 1200,
  currency: "USD",
  transaction_id: "TXN987654321"
});
