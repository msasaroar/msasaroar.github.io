Introduction

Google Tag Manager (GTM) variables are used to dynamically capture values from your website to help manage your tags, triggers, and conditions. There are two main types of variables in GTM: Built-in Variables and User-defined Variables. Below is a breakdown of each type along with examples.

1. Built-in Variables

These are default variables provided by GTM which you can enable as needed.

Variable NameDescriptionExample
Click IDID of the clicked elementbutton123
Click ClassesCSS classes of the clicked elementbtn primary
Page PathCurrent URL path/checkout/step1
ReferrerURL of the referring pagehttps://google.com

2. User-defined Variables

These variables are custom-defined to capture specific values from the site. Here's a list of common types:

Variable TypeDescriptionExample Usage
Data Layer VariableReads values from the dataLayerecommerce.items
URL VariableExtracts values from URL query stringsutm_source
Auto-Event VariableUsed with event-based triggersClick Element
JavaScript VariableReturns value using JavaScriptfunction() { return window.location.hostname; }
First Party CookieReads cookies stored in browser_ga
DOM ElementReads value from an HTML element#product-id
Regex TableMaps input values using regexMatch domain to environment
Lookup TableMaps values directlyPage Path => Page Type

Conclusion

Variables are essential for effective tag management. By understanding how to configure and apply GTM variables, you can track user behavior and trigger events accurately.
