Event: level_start

This event should be triggered when a user begins a level in a game.

Parameters:

ParameterRequiredDescription
level_nameYesName of the level

Example dataLayer.push:

dataLayer.push({
  event: "level_start",
  level_name: "Forest World 1"
});

Event: level_up

This event is used when a user levels up.

Parameters:

ParameterRequiredDescription
levelYesNew level number
characterNoThe character that leveled up

Example dataLayer.push:

dataLayer.push({
  event: "level_up",
  level: 5,
  character: "Warrior Elf"
});

Event: level_end

This event should be sent when a user finishes a level.

Parameters:

ParameterRequiredDescription
level_nameYesName of the level
successNoBoolean: true if level completed successfully

Example dataLayer.push:

dataLayer.push({
  event: "level_end",
  level_name: "Forest World 1",
  success: true
});
