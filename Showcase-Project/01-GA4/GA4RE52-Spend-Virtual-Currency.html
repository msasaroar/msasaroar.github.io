GA4 Event: spend_virtual_currency

Event Definition

This event tracks when a user spends virtual currency in your app or game.

Required Parameters

  • virtual_currency_name – Name of the currency used (e.g., "gems")
  • value – Number of units spent
  • item_name – What the user bought

Data to be pushed:

dataLayer.push({
  event: "spend_virtual_currency",
  virtual_currency_name: "gems",
  value: 200,
  item_name: "Legendary Sword"
});
