GA4 Event: share

Event Definition

The share event logs when a user shares content via social, email, or copy link.

GA4 Parameters

Parameter Type Required Description
content_type string Yes Type of content being shared (e.g., "article")
item_id string Yes Unique ID of the content
method string Yes Sharing method (e.g., "facebook", "email")

Push Data to GTM

dataLayer.push({
  event: "share",
  content_type: "article",
  item_id: "blog-2025-001",
  method: "facebook"
});
