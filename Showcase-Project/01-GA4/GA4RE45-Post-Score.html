GA4 post_score Event Tracking

GA4 Event: post_score

Purpose

Track when a user achieves a score in a game.

Required Parameters

score: 8500
level: "Level 3"
character: "Archer"

DataLayer Push Code

<script>
  dataLayer.push({
    event: "post_score",
    score: 8500,
    level: "Level 3",
    character: "Archer"
  });
</script>
