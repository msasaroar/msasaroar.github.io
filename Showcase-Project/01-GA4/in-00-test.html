view_promotion Event Tracking

GA4 Event: view_promotion

Purpose: Track when a user views a promotion (e.g. banner).

Required Parameters:

  • promotion_id — (String) ID of the promotion.
  • promotion_name — (String) Name of the promotion.
  • creative_name — (String) Name of the creative asset.
  • creative_slot — (String) Name of the location where the promotion is shown.

Sample Data:

{ event: "view_promotion",
ecommerce: {
  promo_view: {
    promotions: [
      {
        promotion_id: "SPRING_SALE",
        promotion_name: "Spring Collection Banner",
        creative_name: "spring_banner_1",
        creative_slot: "home_top"
      }
    ]
  }
}
}
