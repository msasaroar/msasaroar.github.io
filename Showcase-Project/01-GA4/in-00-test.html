View Cart Event Tracking

GA4 Recommended Event: view_cart

Event Definition: Triggered when a user views items in their shopping cart.

Parameters Required:

  • currency – ISO 4217 currency code (e.g., "USD")
  • value – Total value of items in the cart
  • items – Array of items with identifiers and quantities

Example Data Pushed to dataLayer:


dataLayer.push({
  event: "view_cart",
  currency: "USD",
  value: 59.97,
  items: [
    {
      item_id: "SKU_123",
      item_name: "Blue T-Shirt",
      quantity: 1,
      price: 19.99
    },
    {
      item_id: "SKU_456",
      item_name: "Green Hat",
      quantity: 2,
      price: 19.99
    }
  ]
});
