GA4 Event: join_group

This event is triggered when a user joins an online group or community.

Event Parameters

Parameter Required Description
group_id Yes Unique ID of the group
group_name Yes Name of the group

Example DataLayer Push

dataLayer.push({
  event: 'join_group',
  group_id: 'discord123',
  group_name: 'Web Dev Learners'
});
