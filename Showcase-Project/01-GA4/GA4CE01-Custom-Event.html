Page Functionality Overview

This page is a part of a GTM Showcase Project demonstrating how Custom HTML Tags can be used to control button interactions without any JavaScript code in the HTML file itself.

✔️ What Works:

  • Two buttons are displayed: Button-1 and Button-2.
  • Clicking each button triggers a separate alert() message.
  • No JavaScript is written in the HTML file — the behavior is injected via GTM.
  • Demonstrates how GTM can interact with page elements dynamically.

❌ What Does Not Work:

  • No Google Analytics (GA4) events are triggered on button clicks.
  • If GTM is disabled or blocked, the buttons will not perform any action.
  • These buttons do not collect or send any data — they only show alerts.
