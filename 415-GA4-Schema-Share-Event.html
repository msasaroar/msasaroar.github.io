GA4 Share Event Schema

Schema Details

Parameter Name Data Type Description
platform String The platform where the content is shared (e.g., "Facebook", "Twitter").
content_id String The unique identifier for the content being shared.
user_id String The unique identifier for the user.

Example JSON for the Share Event

{
  "event_name": "share",
  "parameters": {
    "platform": "Facebook",
    "content_id": "content123",
    "user_id": "user123"
  }
}
