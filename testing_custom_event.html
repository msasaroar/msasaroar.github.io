Configuaring Custom Event Without GTM

Click the button below to trigger a custom event in GA4.

To track a Button Click custom event in GA4, follow these steps. This event can be configured directly within the GA4 interface, so no additional code is required on the HTML page.

Step 1: Create a Custom Event in GA4

  • Log in to GA4 and select your project.
  • Go to the Admin menu.
  • Under the Events section, click on Create Event.
  • On the Create Event page, click the Create button for Custom Events.
  • Fill in the following details to create the event:
    • Custom Event Name: button_click (this is the name of the event you want to track)
    • Matching Conditions: Set conditions so GA4 can correctly recognize this event.
    • Parameter: Select click or click_text to use the button's text or ID.
    • Operator: equals
    • Value: The text on the button (e.g., "Click Me!" or another value).
  • If you want to add additional parameters like event_category, event_label, and value, configure these under Parameter Configuration. This helps with tracking and reporting.
  • After everything is configured, click Create.

Step 2: Test with GA4 DebugView

  • Navigate to the DebugView option in GA4.
  • Click the button on your website to verify that the event appears in DebugView.
  • If the custom event is working correctly, it will show as button_click and will be trackable in reports.

Step 3: View the Custom Event in GA4 Reports

  • Go to Reports > Events in GA4 to view your custom event and ensure that the button_click event is being tracked.
  • From here, you can analyze your custom event in detail.
