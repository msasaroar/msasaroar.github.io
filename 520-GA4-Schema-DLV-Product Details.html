Data Layer Variable: Product Details

Schema Details

Field Name Data Type Description
product_id String The unique identifier of the product.
product_name String The name of the product.
price Float The price of the product.
currency String The currency of the product price (e.g., "USD").
category String The category to which the product belongs.
brand String The brand of the product.
variant String The variant of the product (e.g., color, size).
quantity Integer The quantity of the product.
