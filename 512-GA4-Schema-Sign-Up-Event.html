GA4 Sign Up Event Schema

Schema Details

Parameter Name Data Type Description
method String The method used for sign up (e.g., "Email", "Google").
user_id String The unique identifier for the user.
success Boolean Indicates whether the sign-up was successful or not.

Example JSON for the Sign Up Event

{
  "event_name": "sign_up",
  "parameters": {
    "method": "Email",
    "user_id": "user123",
    "success": true
  }
}
