GA4 Data Layer Comparison

Data Layer : Compliant To GA4 Schema

This is an example of a data layer structured to meet the GA4 schema requirements.

{
  "event": "purchase",
  "ecommerce": {
    "transaction_id": "T123456",
    "value": 299.99,
    "currency": "USD",
    "items": [
      {
        "item_name": "Laptop",
        "item_id": "LPT123",
        "price": 299.99,
        "quantity": 1,
        "category": "Electronics"
      }
    ]
  }
}
