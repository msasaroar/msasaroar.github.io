GA4 Downloads Event Schema

Schema Details

Parameter Name Data Type Description
file_name String The name of the file downloaded.
file_extension String The file extension of the downloaded file (e.g., ".pdf", ".jpg").
file_size Integer The size of the file in bytes.
download_url String The URL from which the file was downloaded.
page_title String The title of the page where the download occurred.
page_location String The URL of the page where the download occurred.

Example JSON for the Downloads Event

{
  "event_name": "download",
  "parameters": {
    "file_name": "report.pdf",
    "file_extension": ".pdf",
    "file_size": 1048576,
    "download_url": "https://example.com/files/report.pdf",
    "page_title": "Reports Page",
    "page_location": "https://example.com/reports"
  }
}
