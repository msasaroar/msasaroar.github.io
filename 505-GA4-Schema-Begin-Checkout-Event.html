GA4 Begin Checkout Event Schema

Schema Details

Parameter Name Data Type Description
value Float The total value of the items in the checkout.
currency String The currency used (e.g., "USD").
coupon String The coupon code applied at checkout, if any.
items Array An array of items in the checkout, where each item is an object containing details.

Items Parameter Details

Field Name Data Type Description
item_id String The unique identifier of the item.
item_name String The name of the item.
quantity Integer The number of units in the checkout for this item.
price Float The price of a single unit of the item.
item_brand String The brand associated with the item.
item_category String The category associated with the item.
item_variant String The variant of the item (e.g., color or size).
discount Float The discount applied to the item, if any.
