GA4 View Item Event Schema

Schema Details

Parameter Name Data Type Description
currency String The currency used (e.g., "USD").
value Float The value of the item being viewed.
items Array An array of items being viewed, where each item is an object containing details.

Items Parameter Details

Field Name Data Type Description
item_id String The unique identifier of the item.
item_name String The name of the item.
item_brand String The brand associated with the item.
item_category String The category associated with the item.
item_variant String The variant of the item (e.g., color or size).
price Float The price of the item.

Example JSON for the View Item Event

{
  "event_name": "view_item",
  "parameters": {
    "currency": "USD",
    "value": 59.99,
    "items": [
      {
        "item_id": "P001",
        "item_name": "Smartphone",
        "item_brand": "BrandX",
        "item_category": "Electronics",
        "item_variant": "Blue",
        "price": 59.99
      }
    ]
  }
}
