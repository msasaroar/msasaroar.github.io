GA4 Coupon Applied Event Schema

Schema Details

Parameter Name Data Type Description
coupon String The code of the applied coupon.
value Float The value of the discount applied by the coupon.
currency String The currency used for the coupon (e.g., "USD").
items Array An array of items that the coupon is applied to, if applicable.

Items Parameter Details

Field Name Data Type Description
item_id String The unique identifier of the item the coupon is applied to.
item_name String The name of the item the coupon is applied to.
item_quantity Integer The quantity of the item the coupon is applied to.
item_price Float The price of a single unit of the item.

Example JSON for the Coupon Applied Event

{
  "event_name": "coupon_applied",
  "parameters": {
    "coupon": "SAVE10",
    "value": 10.00,
    "currency": "USD",
    "items": [
      {
        "item_id": "P001",
        "item_name": "Laptop Sleeve",
        "item_quantity": 1,
        "item_price": 20.00
      }
    ]
  }
}
