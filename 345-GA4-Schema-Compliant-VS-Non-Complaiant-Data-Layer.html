GA4 Data Dictionary

1. Standard Events

Event Name Description Parameters
page_view Tracks each page load. page_location, page_title, page_referrer
session_start Tracks when a user session starts. engagement_time_msec, session_id
user_engagement Tracks user activity during the session. engagement_time_msec
purchase Tracks completed transactions. transaction_id, value, currency, items

2. Enhanced Measurement Events

Event Name Description Parameters
file_download Triggered on file downloads. file_name, file_extension
outbound_click Triggered on external link clicks. link_url
site_search Tracks internal site searches. search_term, category

3. Standard Parameters

Parameter Name Description Example Values
page_location URL of the current page. https://example.com/about
currency Currency used for monetary values. USD, EUR
value Value associated with the event. 99.99

4. User Properties

Property Name Description Example Values
user_id Unique identifier for the user (set by you). 12345
user_language Language preference of the user. en-US

5. E-Commerce Schema

Transaction Fields

Field Name Description Example Values
transaction_id Unique ID for the transaction. T12345
currency Currency used in the transaction. USD
value Total value of the transaction. 150.00

Items Object Schema

Field Name Description Example Values
item_id Unique identifier for the item. SKU12345
item_name Name of the item. Wireless Mouse
