GA4 Form Submission Event Schema

Schema Details

Parameter Name Data Type Description
form_id String The unique identifier for the form being submitted.
form_name String The name of the form being submitted.
form_action String The action that was performed on the form (e.g., "submit").
form_type String The type of the form (e.g., "contact", "survey").
form_fields Array An array of objects containing the fields in the form and their values.

Form Fields Parameter Details

Field Name Data Type Description
field_name String The name of the form field.
field_value String The value entered into the form field.

Example JSON for the Form Submission Event

{
  "event_name": "form_submission",
  "parameters": {
    "form_id": "contact_form_1",
    "form_name": "Contact Us",
    "form_action": "submit",
    "form_type": "contact",
    "form_fields": [
      {
        "field_name": "name",
        "field_value": "John Doe"
      },
      {
        "field_name": "email",
        "field_value": "johndoe@example.com"
      }
    ]
  }
}
